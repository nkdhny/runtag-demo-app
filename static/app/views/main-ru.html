Мобильное приложение

Мобильное приложение для Android версии 4.4+ доступно тут (только английская версия на данный момент). Для его установки необходимо разрешить "Непроверенные источники" в настройках устройства (подробности смотри, например, в этой статье)
Приложение работает в две фазы

  • сначала оно детектирует объекты, похожие на маркеры (на устройстве)
  • затем отправляет кандидатов на сервер для дальнейшего анализа
Обычно это занимает около нескольких микросекунд и требует около 1Kb сетевого трафика на изображение.

Важно заметить, что процесс детектирования очень чувствителен к качеству снимков, желательно использовать устройства с разрешением камеры более 10Mpx.

Как работает RunTag

Подробное описание принципов работы и ограничений детектирования RunTag вы найдете в презентации.

Кроме того, доступны демо-видео, показывающие возможный сценарий взаимодействия RunTag с инфраструктурой организатора

  • Демо фотографа показывает, как распознать участников на снимках, доступных в публичных источниках (facebook, вконтакте и т.д.)
  • Демо участника демонстрирует возможности по интегрированию RunTag с личным кабинетом организатора
